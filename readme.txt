== Description ==

Using this small plugin you'll be able to deliver content Around the web.

You can choose what Social Network website include in your post; what layout use to this plugin between:

* Horizontal (default)
* Vertical

Supported Website:

* Facebook
* Twitter
* Friendfeed
* StumbleUpon
* Feed RSS DELIVERY
* Digg
* Google Buzz
* Yahoo Buzz

== Installation ==

Install this plugin is very simple:

1. Upload 'social-network-sharer' folder to the `/wp-content/plugins/` directory
2. Activate  plugin through the 'Plugins' menu in WordPress
3. Edit "SNS Settings" in your Wordpress Dashboard
3. Place:

<?php if(function_exists('sharer')) { sharer(); } ?>

in your single.php (read Usage tab on plugin option page for further details)
